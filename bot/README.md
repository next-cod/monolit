# Монолит — Telegram-бот

Бот-витрина студии бренд-стратегии «Монолит». Дополняет сайт: услуги,
процесс, работы, контакты и заявка на обсуждение проекта прямо в Telegram.

## Возможности

- Приветствие и главное меню (инлайн-кнопки)
- Услуги, процесс работы, избранные работы, «о студии» с цифрами
- Контакты (почта, телефон, адрес)
- Заявка «Обсудить проект» — диалог из трёх вопросов, с пересылкой
  администратору (если задан `ADMIN_CHAT_ID`)
- Команды: `/start`, `/menu`, `/brief`, `/contacts`, `/help`, `/cancel`

## Быстрый старт

1. Создайте бота у **@BotFather**: `/newbot` → задайте имя и username.
   Рекомендуемый username: **@monolit_studio_bot** (если занят — любой свободный,
   например `@monolit_brand_bot`). Скопируйте выданный токен.
2. Настройте конфиг:
   ```bash
   cd bot
   cp .env.example .env
   # откройте .env и вставьте BOT_TOKEN, при желании ADMIN_CHAT_ID
   ```
3. Установите зависимости и запустите:
   ```bash
   npm install
   npm start
   ```
   В консоли появится `Запущен как @ваш_бот` — можно писать боту `/start`.

## Заявки администратору

Чтобы заявки из бота приходили вам в личку: узнайте свой Telegram chat id
(например у **@userinfobot**), впишите его в `.env` как `ADMIN_CHAT_ID`,
и **один раз напишите своему боту** `/start` (Telegram не даёт писать первым).

## Полезное

- Проверка синтаксиса без запуска: `npm run check`
- Тех. прогон без подключения к Telegram: `BOT_DRYRUN=1 npm start`
- Хостинг: бот работает на long polling, ему не нужен публичный URL —
  подойдёт любой VPS, Railway, Render или локальный запуск.
